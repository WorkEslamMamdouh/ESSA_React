# Frontend_React

Initial standalone React + TypeScript shell for frontend extraction.

## Implemented in this bootstrap

- Folder structure for app/router/providers/modules/api
- Core React Router setup with guarded app routes
- Module host architecture and registry placeholder
- Typed API client layer (contracts/endpoints/http/clients)
- Global providers for auth, config, and localization

## Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run preview`

> Legacy assets and legacy module code are intentionally not copied in this stage.
